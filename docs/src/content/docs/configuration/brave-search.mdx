Enable web search capabilities using the Brave Search API.

## Configuration

```toml
[brave_search]
api_key = "BSA..."
```

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `api_key` | string | required | Brave Search API key |

## Getting an API Key

<Steps>

1. **Sign up**

   Go to [Brave Search API](https://brave.com/search/api/) and create an account.

2. **Create an API key**

   In the dashboard, create a new API key.

3. **Configure Ash**

   Add to your config:

   ```toml
   [brave_search]
   api_key = "BSA..."
   ```

   Or use environment variable:

   ```bash
   export BRAVE_SEARCH_API_KEY=BSA...
   ```

</Steps>

## Usage

With web search configured, ask Ash to search:

```
Search for the latest Python release notes
```

The assistant will use web search to find current information.

## Free Tier

Brave Search offers a free tier with:
- 2,000 queries/month
- Basic search features

Paid plans available for higher volume.
