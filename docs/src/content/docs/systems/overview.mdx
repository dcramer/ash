Ash is a Python-based personal assistant with modular subsystems.

## System Components

<CardGrid>
  <Card title="Agent Core" icon="puzzle">
    Orchestrates conversations, context assembly, and tool usage.
  </Card>
  <Card title="LLM Providers" icon="rocket">
    Unified interface for `openai`, `openai-oauth`, and `anthropic` models.
  </Card>
  <Card title="Tools + Sandbox" icon="shield">
    Tool execution with Docker sandbox isolation and runtime controls.
  </Card>
  <Card title="Memory" icon="document">
    Context retention, extraction, and optional semantic retrieval.
  </Card>
  <Card title="Providers" icon="telegram">
    Interfaces like CLI and Telegram.
  </Card>
  <Card title="Vision" icon="picture">
    Inbound image understanding and prompt context injection.
  </Card>
  <Card title="Browser" icon="window">
    Session-scoped browser actions via sandbox or Kernel providers.
  </Card>
  <Card title="Skills" icon="setting">
    Local and external skill discovery/execution.
  </Card>
</CardGrid>

## Tech Stack

| Component | Technology |
|-----------|------------|
| Language | Python 3.12+ |
| CLI | Typer |
| Server | FastAPI + Uvicorn |
| Telegram | aiogram 3.x |
| Config/validation | Pydantic |
| Container runtime | docker-py |
