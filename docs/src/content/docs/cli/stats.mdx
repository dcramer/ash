Show operational stats for Ash state and storage.

## Usage

```bash
uv run ash stats
```

## Notes

- `ash info` is an alias for `ash stats`
- Useful for quick operational visibility into your local Ash home
- Includes memory extraction/verification quality counters aggregated from recent log files
