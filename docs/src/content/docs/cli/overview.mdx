Ash provides a comprehensive CLI for interacting with your assistant.

## Usage

```bash
ash [OPTIONS] COMMAND [ARGS]
```

## Global Options

| Option | Description |
|--------|-------------|
| `--help` | Show help message |
| `--version` | Show version |

## Commands

| Command | Description |
|---------|-------------|
| [`chat`](../chat/) | Interactive chat or single prompt |
| [`serve`](../serve/) | Start the server |
| [`config`](../config/) | Configuration management |
| [`memory`](../memory/) | Memory management |
| [`sessions`](../sessions/) | Session management |
| [`sandbox`](../sandbox/) | Sandbox management |
| [`db`](../db/) | Database operations |
| [`upgrade`](../upgrade/) | Run migrations and checks |
| [`service`](../service/) | Service management |

## Getting Help

Get help for any command:

```bash
ash --help
ash chat --help
ash memory --help
```

## Environment Variables

| Variable | Description |
|----------|-------------|
| `ASH_CONFIG` | Config file path |
| `ANTHROPIC_API_KEY` | Anthropic API key |
| `OPENAI_API_KEY` | OpenAI API key |
| `TELEGRAM_BOT_TOKEN` | Telegram bot token |
| `BRAVE_SEARCH_API_KEY` | Brave Search API key |
| `SENTRY_DSN` | Sentry DSN |
