Manage persistent memories that Ash uses across conversations.

## Usage

```bash
uv run ash memory ACTION [OPTIONS]
```

## Actions

### list

List all memories:

```bash
uv run ash memory list
```

Options:
| Option | Default | Description |
|--------|---------|-------------|
| `--query`, `-q` | | Filter by content |
| `--limit`, `-n` | 20 | Maximum entries to show |
| `--scope` | | Filter by scope: personal, shared, or global |
| `--user`, `-u` | | Filter by owner user ID |
| `--chat` | | Filter by chat ID |
| `--include-expired` | false | Include expired entries |

Search memories by content:

```bash
uv run ash memory list -q "api keys"
```

### add

Add a new memory:

```bash
uv run ash memory add -q "Remember to check logs every morning"
```

Options:
| Option | Description |
|--------|-------------|
| `--query`, `-q` | Memory content (required) |
| `--source`, `-s` | Source label (default: cli) |
| `--expires`, `-e` | Days until expiration |

### remove

Remove a memory by ID:

```bash
uv run ash memory remove --id abc123
```

Options:
| Option | Description |
|--------|-------------|
| `--id` | Memory ID (required, or use --all) |
| `--all` | Remove all entries |
| `--scope` | Filter by scope when using --all |
| `--force`, `-f` | Skip confirmation |

### clear

Remove all memories:

```bash
uv run ash memory clear
```

Options:
| Option | Description |
|--------|-------------|
| `--force`, `-f` | Skip confirmation |

## Examples

Add a preference:

```bash
uv run ash memory add -q "Prefer concise code examples"
```

Search memories by content:

```bash
uv run ash memory list -q "project"
```

List personal memories:

```bash
uv run ash memory list --scope personal --limit 10
```
