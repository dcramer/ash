Manage persistent memories that Ash uses across conversations.

## Usage

```bash
uv run ash memory COMMAND [OPTIONS]
```

## Commands

### list

List all memories:

```bash
uv run ash memory list
```

Options:
| Option | Default | Description |
|--------|---------|-------------|
| `--limit`, `-l` | 50 | Maximum entries to show |

### search

Search memories semantically:

```bash
uv run ash memory search -q "project ideas"
```

Options:
| Option | Description |
|--------|-------------|
| `--query`, `-q` | Search query (required) |
| `--limit`, `-l` | Maximum results |

### add

Add a new memory:

```bash
uv run ash memory add -q "Remember to check logs every morning"
```

Options:
| Option | Description |
|--------|-------------|
| `--query`, `-q` | Memory content (required) |

### remove

Remove a memory by ID:

```bash
uv run ash memory remove --id abc123
```

Options:
| Option | Description |
|--------|-------------|
| `--id` | Memory ID (required) |

### clear

Remove all memories:

```bash
uv run ash memory clear
```

### stats

Show memory statistics:

```bash
uv run ash memory stats
```

## Examples

Add a reminder:

```bash
uv run ash memory add -q "Prefer concise code examples"
```

Find project-related memories:

```bash
uv run ash memory search -q "project"
```

List recent memories:

```bash
uv run ash memory list --limit 10
```
