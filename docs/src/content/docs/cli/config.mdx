Manage Ash configuration.

## Usage

```bash
uv run ash config ACTION [OPTIONS]
```

## Actions

### show

Display current configuration:

```bash
uv run ash config show
```

Options:
| Option | Description |
|--------|-------------|
| `--path`, `-p` | Path to config file |

### validate

Validate configuration file:

```bash
uv run ash config validate
```

Checks:
- TOML syntax
- Required fields
- Type correctness
- API key availability

## Examples

Check configuration:

```bash
uv run ash config validate
```

View current settings (redacts secrets):

```bash
uv run ash config show
```
