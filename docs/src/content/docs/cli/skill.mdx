Manage skill discovery, sources, and validation.

## Usage

```bash
uv run ash skill COMMAND [OPTIONS]
```

## Subcommands

| Command | Description |
|---------|-------------|
| `validate` | Validate a skill file format |
| `list` | List discovered skills |
| `add` | Add and install a skill source |
| `remove` | Remove a skill source |
| `sync` | Sync all configured sources |
| `update` | Update installed repos |
| `sources` | Show configured skill sources |

## Examples

```bash
uv run ash skill add owner/repo
uv run ash skill add owner/repo@v1.0.0
uv run ash skill add ~/my-local-skills
uv run ash skill list
uv run ash skill sources
uv run ash skill sync
uv run ash skill update
uv run ash skill validate ~/.ash/workspace/skills/my-skill
```

## Config Structure

```toml
[skills]
auto_sync = false
update_interval = 24

[[skills.sources]]
repo = "owner/repo"

[[skills.sources]]
path = "~/my-local-skills"

[skills.research]
enabled = true
model = "mini"
```
